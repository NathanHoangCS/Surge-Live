# Surge-Live
Surge Live is a web-based sports and esports prediction platform designed to simulate the excitement of live betting without involving real money. Users are given virtual currency to make predictions on live and upcoming events across traditional sports and esports titles. The platform is inspired by community-driven apps like Stadium Live but is built as a fully risk-free and legally compliant experience focused on competition, strategy, and user engagement rather than cash payouts.

What sets Surge Live apart is its emphasis on long-term progression and performance tracking. Instead of one-off wagers, users level up through consistent accuracy, unlock new features over time, and earn non-monetary rewards such as badges, enhanced analytics, and expanded prediction options. The platform tracks metrics like prediction accuracy, win/loss streaks, and sport-specific performance, encouraging users to improve their decision-making skills over time.
